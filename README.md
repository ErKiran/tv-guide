# tv-guide

A simple TV guide for the terminal.

Availables commands 

```bash
Usage:
  tv-guide [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  movies      List Current and Upcoming Movies in Hindi Channel

Flags:
  -h, --help   help for tv-guide

Use "tv-guide [command] --help" for more information about a command.
```

For movies command 

```bash 
tv-guide movies 
```
